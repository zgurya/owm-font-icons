This plugin allows you to create shortcodes to display the weather information using font icons. Weather information provided by OpenWeatherMap API.

== Description ==

With this plugin, you can:
* Create your weather informers and insert them into the content
* Create widgets
* Use cool fonts icons
* Get data from openweathermap.org

== Installation ==

1. Install via the WordPress plugin admin dialog (search for Forecast Font Icons) or
2. Upload to your plugins folder, usually wp-content/plugins/, keeping the directory structure intact (i.e. owm-font-icons.php should end up in wp-content/plugins/owm-font-icons/).
3. Activate the plugin on the plugin screen.
4. Create informer or add widget to sidebar
